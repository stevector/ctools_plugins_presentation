Zen Grids uses SASS

"Sass is a meta-language on top of CSS…"

source: http://sass-lang.com/about.html

What does SASS do?

  • SASS (SCSS)

    .views-row { 
   border: 1px solid #ccc;
   span {
     color: #400;
   }
 }

  • CSS

    .views-row {
   border: 1px solid #ccc;
 }
 .views-row span {
  color: #400;
}

What else?

  • Variables
    $nw-purple : #520063;
a {
  color: $nw-purple;
}
  • Mixins (kinda like a function)
    @mixin left($dist) {
  float: left;
  margin-left: $dist;
}
  • Selector Inheritance
    .error {
  border: 1px #f00;
  background: #fdd;
}
.badError {
  @extend .error;
  border-width: 3px;
}
  • Functions
    .block-b {
  @include zen-grid-item(1, 1, right);
}
.block-c {
  @include zen-clear(right);
  @include zen-grid-item(1, 1, right);
}
No time for details here. Go to http://sass-lang.com for more information on the first three.
